# 🫀 Black Souls: Console Labyrinth RPG by KET_KAT

Хардкорная текстовая ролевая игра с элементами выживания и уникальной боевой системой, вдохновленной вселенной Black Souls.

## ⚔️ Особенности боевой системы и геймплея
* **Механика AP (Action Points)**: Каждое действие в бою (мощный удар, быстрый выпад, перекат) требует выносливости. Спам атак приведет к истощению, и монстр накажет вас за усталость!
* **Античит-генерация путей**: Направления комнат генерируются случайно при каждом запуске. Игру невозможно заучить наизусть.
* **Тактический блок**: Игрок может снизить урон от гоблина на 13%, вовремя используя щит.
* **Наказание за невнимательность**: Если нажать не ту цифру в меню, гоблин нанесет урон, пока персонаж «тупит».
* **Атмосферный ASCII-арт**: Детализированный череп на экране проигрыша.

## 🛠 Технологии проекта
* Python 3.12.9
* Встроенные модули: `random`, `os`, `time`

## 📦 Запуск игры
Проект не требует установки сторонних библиотек. Скачайте файл с кодом и запустите его через терминал:
```bash
python "console rpg labyrinth.py"
```
